# Sugar AI

A Flutter/Dart mobile application built with AI integration. Cross-platform mobile app for Android and iOS.

## Tech Stack

- Dart, Flutter
- Android SDK

## Getting Started

```bash
git clone https://github.com/SaadRiaz99/Sugar_Ai.git
cd Sugar_Ai
flutter pub get
flutter run
```
